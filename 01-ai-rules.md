# 전 프로젝트 공통 AI 절대 룰 (Global AI Rules)

이 파일은 모든 프로젝트에서 AI가 코딩 및 답변을 할 때 **최우선으로 무조건 지켜야 하는 절대 규칙**입니다.

## 1. 언어 설정
* **모든 답변은 무조건 한글(Korean)로 답변할 것.** (영어 질문/에러가 들어와도 설명은 한글로 제공)

## 2. Python 패키지 설치 규칙
* 모든 파이썬 패키지를 서버나 로컬에 설치(다운로드)할 때는, 터미널 명령어 서두에 항상 `python -m` 명령어를 먼저 붙인 뒤에 `pip`를 이어가도록 할 것.
* ❌ 잘못된 예: `pip install requests`
* ✅ 올바른 예: `python -m pip install requests`
